# Python File Sorter

A simple Python program that automatically organizes files into folders based on their extensions.

## Features

- Sorts images, documents, and other files
- Automatically creates folders
- Helps keep your Downloads folder organized

## Technologies Used

- Python
- pathlib
- shutil

## Future Improvements

- Add a graphical interface
- Allow custom sorting rules
- Add support for more file types
